# 🍁 Canada Citizenship Test Prep

A study app with 500 practice questions, 20 mock tests, and learning streaks.

Built to help newcomers pass the Canadian Citizenship Test on their first try.

## Topics Covered
- Rights & Responsibilities
- Canadian History
- Government & Elections
- Justice System
- Canadian Symbols & Geography
- Indigenous Peoples
- Modern Canada & Economy
- Military & Peacekeeping
- How to Apply for Citizenship

## Features
- 600 questions across 20 sets
- Timed mock tests (20 questions, 30 min)
- Daily learning streak tracking
- Progress saved automatically
- Answer review after each mock test
